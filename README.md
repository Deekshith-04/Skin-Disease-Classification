# Skin Disease Classification using Deep Learning

## Overview

This project uses a CNN model trained on the HAM10000 dataset to classify skin diseases.

## Features

* Image upload & camera scan
* Disease prediction with confidence
* Prescription & precautions
* Image validation (dark/low-quality detection)

## Model

* CNN (TensorFlow/Keras)
* 7 classes: akiec, bcc, bkl, df, mel, nv, vasc

## Tech Stack

* Python
* TensorFlow
* Streamlit
